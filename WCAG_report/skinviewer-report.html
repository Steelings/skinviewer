Executive Summary

Overall readability is consistent, and provides people that use screenreaders a good representation of what the website is about.

Consistent usageof headers and footers using the same tags and redirects are key to get a better understanding of the site.

Frequently occuring patterns:

  1. VR experience is not accessible without mouse-downs
  2. About page does not contain any usuable info of what it is about.
  3. Mobile responsiveness does contain jagged UI sometimes, that frequently occurs on multiple pages.
